# Temperature Sensor DHT11

If you are testing with the C# code don't forget to change the line:

_dht = new Dht11(_pin, GpioPinDriveMode.Input);

to use the Dht22 instead:

_dht = new Dht22(_pin, GpioPinDriveMode.Input);
